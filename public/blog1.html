How I Built My First Website 🌐

I still remember the spark. It was a rainy afternoon in Kathmandu, and I was exploring the internet when I stumbled across a portfolio site that made me whisper to myself, "I want to build something like that one day." That moment became the beginning of a magical coding journey.

🌟 The Spark of Curiosity

I began with absolutely no idea where to start. HTML looked like alien code. CSS was just... confusing. But I loved the idea of controlling pixels on a screen. So I watched YouTube tutorials, took notes like a wizard mastering spells, and coded late into the night.

🛠 My Tools of Magic

  • HTML5 – My site's skeleton, the bones that gave it shape.
  • CSS3 – The colors, the gradients, the smooth animations — the magic wand.
  • VS Code – My digital spellbook. Theme: Dracula.
  • Git & GitHub – Learning to commit like a pro and share with the world.

🎯 The Challenges I Faced

My navbar wouldn't center. My animations were jittery. I accidentally broke my entire layout at least five times. But each bug made me better. I wasn't just building a website — I was building resilience.

"You don’t need to be great to start. But you need to start to become great."

🚀 The Launch

I finally hosted it using GitHub Pages. Hitting that deploy button felt like launching a rocket. My very own space on the internet — live, clickable, and mine.

💡 What I Learned

  • The web is built by dreamers who dare to type.
  • Everything is hard before it becomes easy.
  • Consistency beats intensity. One hour a day can change your life.
